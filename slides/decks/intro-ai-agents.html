Slide 2 · Definition

What is an AI Agent?

An AI agent is a system that perceives its environment, reasons about a goal, and acts — often in multiple steps — to accomplish that goal, without requiring a human to direct every move.
👁

Perceive

Takes in text, data, images, tool output, or environment state as input.

🧠

Reason

Uses a large language model (or other model) to plan and decide next steps.

🔧

Act

Calls tools: runs code, searches the web, reads files, calls APIs.

🔄

Loop

Observes the result and decides whether the goal is met or more steps are needed.

Slide 3 · History

Key Technology Milestones

2012

AlexNet — Deep Learning Breakthrough

Convolutional networks crush image recognition; GPUs become the new AI engine.

2016

AlphaGo beats Lee Sedol

DeepMind's agent masters Go — once thought impossible — via reinforcement learning.

2017

Transformer Architecture

"Attention Is All You Need" — the architecture behind every modern LLM.

2020

GPT-3 & AlphaFold 2

GPT-3 demonstrates few-shot reasoning. AlphaFold solves 50-year protein-folding problem.

2022

ChatGPT — Mass Adoption

100 million users in 2 months. LLMs become a mainstream tool, not just research.

2023

Tool Use & Plugins

Models can call external tools (search, code, APIs) — the first real "agents".

2024

Reasoning Models (o1, R1)

Models that think step-by-step before answering. Math and science tasks near human expert level.

2025–26

Agentic Workflows & Coding Agents

Agents write, run, and debug entire codebases. Scientists co-author papers with AI.

Slide 4 · Practice

How AI Agents Are Used Today

💬

Chat Interfaces

ChatGPT, Claude, Gemini. Conversational, zero-setup. Best for brainstorming, drafting, Q&A, and summarization.

⌨️

CLI & IDE Agents

Claude Code, GitHub Copilot, Cursor. Live inside your terminal or editor. Can read, write, and run entire codebases.

🔌

API & Pipelines

Embed agents in your own apps or research workflows via the Anthropic / OpenAI API. Fully programmable.

🔍

Deep Research Agents

Perplexity, Gemini Deep Research. Autonomously searches, reads sources, synthesizes long reports.

🖥️

Computer-Use Agents

Operate a real browser or desktop (Operator, Claude Computer Use). Fill forms, collect data, navigate sites.

🤝

Multi-Agent Systems

Orchestrator assigns tasks to specialist sub-agents. Used for complex workflows like coding + testing + deployment.

Slide 5 · Applications

AI Agents for Economics:
Teaching & Research

📚

Teaching

Personalized tutoring — students get instant feedback on problem sets and concepts.

Case study generation — create custom exercises from real economic events on demand.

Simulation & role-play — students negotiate with an AI central banker or firm in market games.

Accessibility — explain concepts at different levels; translate materials instantly.

🔬

Research

Literature review — summarize and synthesize dozens of papers; find gaps faster.

Data wrangling & code — write and debug R/Python/Stata scripts; clean messy datasets.

Web & text data collection — scrape, parse, and structure large text corpora for empirical work.

Drafting & editing — iterate on introductions, referee responses, and grant proposals.
