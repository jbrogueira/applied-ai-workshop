# Applied AI Workshop

Workshop materials for teaching and research on applied AI tools.

## Structure

```
applied-ai-workshop/
├── slides/
│   ├── decks/        # Presentation files (.pptx, .pdf, .key, etc.)
│   └── assets/       # Images, diagrams, and media used in slides
├── code/
│   ├── examples/     # Standalone code examples
│   ├── notebooks/    # Jupyter notebooks
│   └── scripts/      # Utility and helper scripts
├── data/
│   ├── raw/          # Original, unmodified datasets
│   ├── processed/    # Cleaned or transformed data
│   └── samples/      # Small sample datasets for demos
└── resources/
    ├── papers/       # Reference papers and articles
    └── references/   # Links, cheat sheets, and other reference material
```
